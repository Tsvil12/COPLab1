# Лабораторная работа №1: Класс Matrix на C++

## Описание
Реализация класса Matrix для работы с матрицами. Поддерживает основные операции: сложение, вычитание, умножение, транспонирование, вычисление определителя.

## Структура проекта
lab1/
├── matrix.hpp # объявление класса
├── matrix.cpp # реализация методов
├── main.cpp # тестирование
└── README.md # описание проект


## Компиляция и запуск

### В MSYS2 UCRT64:

cd /c/Users/pavel/Desktop/COP/lab1
g++ matrix.cpp main.cpp -o lab1.exe -fexec-charset=cp866
./lab1.exe
